# volopay assignment

## Pre-Requisites

  * Node v16.5.0

## Steps to run the application

  * To install dependencies
  
    ```node.js
    yarn
    ```
  * To start metro
  
    ```node.js
    yarn run start
    ```
  * To start your application
  
    ```node.js
    yarn run android
    ```
## To build the application
   
   ```node.js
   yarn run build
   ```
